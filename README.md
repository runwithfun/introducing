# Фитнес Трекер

## Описание
Фитнес Трекер - это многофункциональное мобильное приложение для отслеживания физической активности, включая бег, велосипедные прогулки, плавание и другие виды тренировок. Приложение предоставляет подробную статистику, маршруты, анализ сердечного ритма и многое другое.

## Текущие вызовы и решения

### Проблема: Неоптимальное хранение данных
**Описание:** Для анализа и отображения изменений показателей, собранных из тренировок за всё время, код постоянно заново к ним обращается и обновляет их. Это приводит к избыточным запросам и замедляет работу приложения.

**Решение:** Пересмотреть сбор данных из тренировок, чтобы агрегированные данные хранились в кэше и можно было быстро к ним обращаться. Придумать как собирать сразу все показатели из тренировок, чтобы не фетчить по отдельности.

### Проблема: Избыточный вызов функций
**Описание:** Некоторые функции вызываются даже на страницах, где они не используются, что приводит к ненужной нагрузке на устройство и потенциальным проблемам с производительностью.

**Решение:** Пересмотреть логику вызова функций, реализовать ленивую загрузку компонентов и функций. Использовать паттерны проектирования, которые позволят загружать и выполнять только те функции, которые действительно необходимы на текущей странице.

## Анализ конкурентов

Мы внимательно изучаем рынок фитнес-приложений, чтобы предложить нашим пользователям лучшие решения. Вот как наш Фитнес Трекер соотносится с основными конкурентами:

### Основные конкуренты и их особенности

#### 1. Strava
**Плюсы:**
- Сильное сообщество и социальные функции (сегменты, соревнования)
- Детальный анализ маршрутов и скорости
- Интеграция с большинством устройств (Garmin, Fitbit, Apple Watch)
- Функция "Beacon" для безопасности

**Минусы:**
- Многие продвинутые функции требуют платной подписки
- Менее детальный анализ сердечного ритма по сравнению с нашим приложением
- Ориентирован в основном на бег и велоспорт

#### 2. Garmin Connect
**Плюсы:**
- Очень детальная аналитика тренировок
- Интеграция с экосистемой устройств Garmin
- Расширенные функции для бега с планами тренировок
- Точное отслеживание сна и восстановления

**Минусы:**
- Сложный интерфейс, менее интуитивный
- Ограниченная социальная составляющая
- Наибольшую пользу приносит только с устройствами Garmin

#### 3. Nike Run Club
**Плюсы:**
- Бесплатные планы тренировок от профессионалов
- Аудио-сопровождение тренировок
- Простой и стильный интерфейс
- Система достижений и наград

**Минусы:**
- Ориентирован преимущественно на бег
- Ограниченная аналитика данных
- Меньше возможностей для отслеживания других активностей
- Ограниченная интеграция с устройствами

#### 4. Fitbit
**Плюсы:**
- Комплексное отслеживание здоровья (сон, стресс, женское здоровье)
- Интеграция с весами и другими устройствами Fitbit
- Отслеживание питания и воды
- Сильное сообщество и вызовы

**Минусы:**
- Полный функционал доступен только с устройствами Fitbit
- Требуется подписка Premium для расширенной аналитики
- Менее точные карты маршрутов
- Ограниченный функционал без подписки

#### 5. Apple Fitness+
**Плюсы:**
- Глубокая интеграция с экосистемой Apple
- Высококачественные тренировки с видео
- Плавная синхронизация между устройствами
- Интуитивно понятный дизайн

**Минусы:**
- Работает только на устройствах Apple
- Требует платной подписки
- Меньше функций для серьезных спортсменов
- Ограниченная социальная составляющая

### Наши преимущества

Анализируя конкурентов, мы выделили ключевые преимущества нашего Фитнес Трекера:

- **Детальный анализ сердечного ритма по зонам** — более подробный, чем у большинства конкурентов, с разбивкой на 5 зон и процентным соотношением
- **Универсальность** — хорошая поддержка разных видов активности без явной специализации
- **Подробная визуализация данных** — годовая и ежедневная статистика представлена наглядно
- **Низкие системные требования** (iOS 12.0+) — доступность для пользователей старых устройств
- **Простой интерфейс** с интуитивно понятной навигацией
